# 台灣 2021 COVID-19 Alpha 波段：流行病學模型與反事實模擬分析報告

## 摘要 (Executive Summary)
本研究旨在分析 2021 年 5 月至 8 月台灣爆發 COVID-19 Alpha 變異株期間，非藥物介入措施 (NPIs，即三級警戒) 的防疫成效。透過 **SEIR 傳染病模型** 與 **合成對照法 (Synthetic Control Method)**，我們量化了政策介入對遏止病毒傳播的關鍵影響。結果顯示，早期的三級警戒成功將基本傳染數 ($R_0$) 壓制至 0.94；若措施延遲兩週，確診數恐增加近 3 倍；若無任何作為，全台將有超過 320 萬人感染。

我們使用擴充的 SEIR (Susceptible-Exposed-Infectious-Recovered) 模型來擬合當時的確診數據。

*   **分析期間**：2021-05-01 至 2021-08-31
*   **關鍵參數估計**：
    *   **初始感染數 ($I_0$)**：約 1,572 人 (推估當時社區中的隱形傳播鏈)
    *   **傳染率 ($\beta$)**：約 0.0942
    *   **基本再生數 ($R_0$)**：**0.94**

> 數據顯示，$R_0 < 1$ (0.94) 代表當時嚴格的三級警戒措施有效阻斷了病毒擴散，使疫情在 3 個月內獲得控制。

## 2. 平行世界：反事實模擬 (Counterfactual Scenarios)
為了回答「如果晚一點封城會怎樣？」的問題，我們建立了三個模擬情境：

| 情境 | 描述 | 關鍵假設 |
| :--- | :--- | :--- |
| **A: 現實世界 (Reality)** | 實際發生的歷史 | 5/15 實施 NPIs ($R_0 \approx 0.94$) |
| **B: 延遲介入 (Late NPIs)** | 假設延後 **2週** 才升三級 | 高傳播率持續額外 14 天後才下降 |
| **C: 佛系防疫 (No NPIs)** | 完全不實施管制 | 維持初始高傳播率 ($R_0 \approx 2.5$) |

### 模擬結果分析

2.  **情境 B (延遲 2 週)**：僅僅晚兩週行動，累積確診數將暴增至 **10,827** 例。這近 **3倍** 的差距顯示了指數成長期「時間」的巨大價值。
3.  **情境 C (無作為)**：若無任何防疫措施，模型預測疫情將完全失控，累積確診數將突破 **320 萬** 例，徹底擊垮台灣的醫療體系。

## 3. 因果推論：雙北的超額衝擊
除了 SEIR 模型，我們亦採用 **合成對照法 (Synthetic Control Method, SCM)**，以數據驅動的方式找出雙北地區（疫情熱區）的對照組。

*   **實驗組**：台北市
*   **對照組池**：台灣其他縣市
*   **分析結果**：演算法識別出 **彰化縣** 為最佳的合成對照對象 (權重 $\approx$ 1.0)，意即在疫情爆發前，彰化縣的公衛特徵與台北市最為相似。
*   **衝擊量化**：在 5/15 之後，台北市的實際案例數急劇偏離了合成彰化的基線。這段「差距 (Gap)」即代表了該次群聚感染事件對都會區造成的具體超額衝擊 (Excess Cases)。
